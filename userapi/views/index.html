Welcome to the DevOps Project!

Node.js

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine, ideal for building fast, scalable network applications.

Next.js

Next.js is a React framework that enables functionality such as server-side rendering and generating static websites for React based web applications.

Docker

Docker is a set of platform as a service products that use OS-level virtualization to deliver software in packages called containers.

Vagrant

Vagrant is an open-source software product for building and maintaining portable virtual software development environments.

Kubernetes

Kubernetes is an open-source container-orchestration system for automating computer application deployment, scaling, and management.

Istio

Istio is an open-source service mesh that provides a way to control how microservices share data and resources.

Prometheus

Prometheus is an open-source systems monitoring and alerting toolkit originally built at SoundCloud.

Grafana

Grafana is an open-source platform for monitoring and observability. It allows you to query, visualize, alert on, and understand your metrics no matter where they are stored.
